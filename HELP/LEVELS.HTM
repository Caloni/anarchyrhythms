Level Sliders

The input slider controls how much input signal goes into AnarchyRhythms. The level of internal oscillation (if oscillators are being used) is not affected, so
you can use this to balance input against internal oscillation.		   The output slider controls how much output comes out of AnarchyRhythms.

The dry/wet slider crossfades between dry (the input signal) and wet (AnarchyRhythms' output signal).

Adjust sliders by left clicking on them.
