Pattern Editor

Sequencing Controls

These are:
  • Host syncronisation. Overrides internal Bpm and locks AnarchyRhythms to the timing of the host sequencer.
  • Bpm. The speed of the pattern measured in beats per minute.
  • Beats. The number of beats in the sequence.
  • Substeps. The number of subdivisions within each beat.

Left click on numeric values to decrease them, right click to increase them. Hold down the <ctrl> key at the same time to fast-adjust.
