Menu Commands

Presets Menu

AnarchyRhythms supplies a collection of preset patterns. These are a good way to quickly familiarise yourself with the sort of things AnarchyRhythms does.

The presets menu also contains an option to import patches from older versions of AnarchyRhythms.

Sound Menu

Preset sounds

AnarchyRhythms also supplies a collection of individual preset sounds, which can be dropped into an existing pattern .

Load/Save user sound

Use these to transfer individual sounds between different patterns, or to build your own collection of sounds.

Note that when using preset sounds, or when loading and saving user sounds, only the highlighted channel is refered to. This menu is for dealing with one channel in isolation. If you want to load or save an entire pattern, use the load effect/save effect options supplied by the host application.

Midi Map Menu

Edit Midi Automation Map

Opens the Midi Automation Dialog. This dialog allows you to choose a midi channel and connect parameters to midi control and pitchbend messages.

Load/Save Midi Automation Map

These commands allow you to load and save midi settings, independently of each saved 'effect'. Use them when you want to transfer automation settings from one pattern to another.

Window Size Menu

You can make AnarchyRhythms big or small!

Help Menu

Links to the help you are now reading.

About box, containing registered user's serial number.

Links to AnarchySoundSoftware

Anarchy Button

Randomises the current patch. This is a quick way to get ideas!

Context Menu

Right click on AnarchyRhythms, wherever you click, you will be presented with a menu which is relevent to where you clicked.

There is context sensitive help, and parameter mapping, and access to the 'sound' menu.
