AnarchyRhythms Installation Notes

  • Your VST Plugin directory will normally be VstPlugins directory in Program Files\Steinberg.
  • Some VST host applications allow you to change VstPlugins directory (or have multiple VstPlugins directories). If a host doesn't recognise AnarchyRhythms, explore its menus for an option to set the VstPlugins directory. Then, either change the host's VstPlugins directory to where you installed AnarchyRhythms, or re-install AnarchyRhythms into the hosts current VstPlugins directory.
  • Unfortunately some host applications require the plugins to be installed into a VstPlugins directory local to them. If you are having trouble getting a host application to recognise AnarchyRhythms, try installing it into a VstPlugins directory within the program directory of that host.
  • The Standalone .exe version needs an 'application directory'. You can install it anywhere, a good place would be Program Files\AnarchySoundSoftware\AnarchyRhythms. This HELP file installs into the same directory.
