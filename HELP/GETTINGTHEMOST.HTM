Getting the most from AnarchyRhythms

Advice about input sounds

AnarchyRhythms sounds best when the input sound contains a broad frequency spectrum, as this gives the filters more to get their teeth into. Examples of sounds with broad frequency spectrums:

  • Resonant or noisy sounds
  • Mixed layers of differently pitched sounds
  • Sounds which change in pitch over their duration

If you want to pre-process sounds to broaden their frequency spectrums, try using a resonator (such as AnarchyEffects HarmonicAdder, published by Anarchy Sound Software.

AnarchyRhythms also can sound better when the input sound has a rhythmic metre matching that of the AnarchyRhythms pattern (ie same BPM and time-signature), or when the input sound is largely static (ie not much rhythmic metre of its own).

Getting a matching rhythmic metre:

Do this by setting the bpm on AnarchyRhythms to match that of the input sound, or if the host application is a sequencer, select the Host Sync button on the Sequencing Controls.

Examples of static input sounds (sounds with little rhythmic metre):

  • Soundscapes
  • Sustained notes
  • Constant-velocity streams of notes
  • Reverbs
  • Massively-slowed time stretch sounds

If you want to pre-process sounds to increase their staticness, try using reverb or echo effects to blur their rhythmic metre.

Advice about parameter settings

If an AnarchyRhythms sound is messy or unclear, here are some things to try

  • Decrease its length. Shorter sounds are always crisper.
  • Increase its feedback amount. Feedback increases consistency from one sound to the next, and masks irregularities in the input signal.
  • Increase oscillator level. AnarchyRhythms internal oscillators cover over any messiness altogether.

If AnarchyRhythms is producing clicks, check for discontinuities in the edit pane envelope graphs. Discontinuities are vertical lines in the graph - these make envelope values change instantaneously which causes audible clicks. Instead use a gradient, however steep it is doesn't matter so long as its not vertical!
