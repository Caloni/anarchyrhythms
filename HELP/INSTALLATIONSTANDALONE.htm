AnarchyRhythms

Standalone .exe Installation Notes

The Standalone version needs an 'application directory'. You can install it anywhere, a good place would be Program Files\AnarchySoundSoftware\AnarchyRhythms. This HELP file installs into the same directory.

The Standalone version allows you to set its inputs, outputs, and streaming parameters.

The first time you run it, you will presented with this configuration dialog. The dialog can be reopened at any time, by selecting the 'file' menu, then the 'configure' option.

If you select smaller buffer sizes, you will get less signal latency (delay). If the buffer is too small you will get clicks and other digital interference in your sound.

The optimum buffer size will depend on your soundcard, system, etc so experiment!
