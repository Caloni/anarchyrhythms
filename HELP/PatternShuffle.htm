Pattern Editor

Shuffle Controls

These are:
  • Shuffle Mode. Change this by clicking on the 'shuffle' button, and choosing to shuffle by 'step' or 'beat'. Step shuffling elongates and shortens substeps alternately. Beat shuffling enolgates the first substep of each beat, and shortens subsequent substeps accordingly.
  • Shuffle Slider. This sets the extent of shuffle. At the far left position there is straight time, the far right position is highly swung. Note that as this parameter changes, the pattern editor scales itself to reflect the current shuffle setting.
