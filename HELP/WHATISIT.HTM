Welcome to AnarchyRhythms

AnarchyRhythms is a hybrid between an effect and a drum machine.

It takes an audio input and applies a user-controlled matrix of amplitude modulation, bandpass filtering, comb filters, feedback loops, internal oscillation and compression.

This allows you to compose rhythmic patterns from the sonic timbre of the input.

  • Pattern editor for composition of rhythmic patterns
  • Multiple Patterns, with automatable switching between patterns
  • Mixer to control parameters for each of four sound channels
  • Edit pane to specify envelopes for the parameters
  • Collection of preset patterns and sounds
  • Parameter automation using MIDI controllers or VST automation
  • VST Time Information to synchronise with host sequencers
  • Supports non-4/4 time signatures
  • Analog Shuffle Functionality.
  • VST and Standalone Executable versions
  • Context-sensitive help system
  • Online support also available
